## Que es

Analisis Estrategico IA es la vista ejecutiva del sistema para consolidar el estado del SGC, detectar brechas, priorizar decisiones y proponer acciones confirmables. Comercialmente se presenta como Centro de Inteligencia Gerencial.

## Para que sirve

Sirve para transformar datos dispersos del sistema en una lectura gerencial unica. Resume score global, brechas normativas, hallazgos, alertas, oportunidades y prioridades 30/60/90. Tambien permite preparar acciones via DirectAction y generar una base para revision por la direccion.

## Como se usa

1. Ingresar a **Analisis estrategico** desde el menu de Direccion o desde `mi-panel`.
2. Abrir **Nuevo analisis** y elegir el alcance: organizacional, proceso, puesto, persona o revision por la direccion.
3. Si el alcance lo requiere, completar la entidad objetivo con tipo, ID y nombre.
4. Definir orientacion de lectura y horizonte temporal.
5. Generar el informe y revisar el resumen ejecutivo, el score, las brechas y las prioridades.
6. Usar **Solicitar DirectAction** para crear una accion confirmable sobre una sugerencia.
7. Usar **Usar como base para revision por la direccion** cuando el informe deba convertirse en insumo formal de direccion.

## Errores frecuentes

- Generar un analisis por proceso o por persona sin cargar correctamente la entidad objetivo: el contexto puede salir incompleto.
- Interpretar el informe como accion ejecutada: las acciones sugeridas no se ejecutan solas, requieren confirmacion humana.
- Usar el modulo sin datos base en auditorias, acciones, objetivos o indicadores: el score y las prioridades pierden calidad si el sistema no esta alimentado.
