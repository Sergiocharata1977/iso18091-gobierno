## Que es

El Panel Ejecutivo centraliza los indicadores de alto nivel del sistema de gestion: cumplimiento de objetivos, estado de auditorias, hallazgos abiertos y tendencias de mejora. Tambien conecta con Analisis Estrategico IA como capa ejecutiva persistida. Es el punto de entrada para la revision por la direccion.

## Para que sirve

Permite a la alta direccion evaluar la eficacia del SGC sin necesidad de ingresar a cada modulo. Facilita la toma de decisiones estrategicas con datos actualizados y con informes consolidados del Centro de Inteligencia Gerencial.

## Como se usa

1. Ingresar a **Panel ejecutivo** desde el menu lateral.
2. Revisar el estado global: objetivos cumplidos, auditorias activas y hallazgos sin cerrar.
3. Usar **Indicadores de madurez** (`/maturity`) para evaluar el nivel de implementacion ISO.
4. Usar **Analisis estrategico** (`/analisis-estrategico`) para generar una lectura consolidada y persistida del sistema.
5. Usar **Planificacion** (`/planificacion-revision-direccion`) para registrar las revisiones formales por la direccion o crear una base desde el ultimo analisis.
6. Exportar o compartir resultados con el equipo directivo.

## Errores frecuentes

- Usar el panel sin haber cargado objetivos de calidad: los KPIs apareceran vacios.
- No actualizar el estado de auditorias y hallazgos: el panel refleja datos en tiempo real, si los modulos no estan al dia el panel tampoco lo estara.
- Confundir panel ejecutivo con analisis persistido: el panel resume, pero el informe formal se genera desde `Analisis estrategico`.
