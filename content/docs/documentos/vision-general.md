## Que es

El modulo de Documentos centraliza procedimientos, instructivos, formularios, registros y politicas del sistema de gestion.

## Para que sirve

Sirve para asegurar que cada persona use la version correcta, que los cambios queden trazados y que los documentos esten vinculados con procesos, areas o requisitos aplicables.

## Como se usa

1. Ingresar a `Documentos` y filtrar por tipo, area o proceso.
2. Crear o editar el documento segun permisos.
3. Cargar metadatos basicos como codigo, titulo, categoria y relacion con norma o proceso.
4. Enviar el documento a revision y luego a aprobacion.
5. Consultar el historial cuando se necesite justificar cambios o recuperar versiones anteriores.

## Errores frecuentes

- Compartir archivos fuera del modulo y luego olvidar actualizarlos en la version oficial.
- Aprobar documentos sin categoria o sin relacion clara con el proceso que regulan.
- Usar un registro obsoleto porque no se reviso el estado antes de descargarlo.
