## Que es

El Centro de Mensajes es la bandeja de entrada unificada que reune todas las conversaciones del sistema: chats con el asistente IA Don Candido y conversaciones de WhatsApp Business (cuando el plugin esta habilitado).

## Para que sirve

Permite ver en un solo lugar todas las interacciones con clientes, empleados o el asistente de IA, sin tener que revisar por separado el chat IA y el inbox de WhatsApp.

## Como se usa

1. Ingresar a **Mensajes** desde el menu lateral (acceso rapido).
2. Ver el listado de conversaciones ordenadas por actividad reciente.
3. Filtrar por canal: IA o WhatsApp.
4. Abrir una conversacion para ver el historial completo y agregar notas internas.
5. Las notas internas son visibles solo para el equipo, no para el cliente.

## Errores frecuentes

- Confundir el Centro de Mensajes con el chat IA flotante: son complementarios. El chat IA es para consultas rapidas; el Centro de Mensajes es para gestion de conversaciones.
- Ver conversaciones vacias de WhatsApp: verificar que el numero este conectado en Configuracion > WhatsApp.
