## Que es

El ABM de declaraciones permite registrar manifestaciones formales, constancias o datos estructurados que complementan la evidencia del modulo Mejoras.

## Para que sirve

Sirve para dejar trazabilidad documental cuando una auditoria, un hallazgo o una accion requiere una declaracion asociada antes del cierre o la verificacion.

## Como se usa

1. Entrar a `Mejoras > Declaraciones`.
2. Revisar el listado y usar filtros si hay varios registros.
3. Crear una nueva declaracion desde `Nueva`.
4. Completar responsable, contexto y contenido de la declaracion.
5. Volver al detalle para editar, validar o relacionar el registro con el seguimiento correspondiente.

## Errores frecuentes

- Crear declaraciones sin indicar a que caso o evidencia responden.
- Guardar texto preliminar como version final sin revisarlo con el responsable.
- Duplicar declaraciones cuando el mismo caso ya tiene una constancia previa.
