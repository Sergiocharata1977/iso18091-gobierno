## Que es

CRM es la herramienta de gestion comercial para clientes, oportunidades y analisis, mientras que la app de vendedores resuelve la operacion de campo desde celular o tablet.

## Para que sirve

Sirve para separar trabajo estrategico y trabajo operativo: el CRM web permite analizar cartera, pipeline y riesgo; la app movil permite registrar visitas, evidencias y tareas aun sin conectividad.

## Como se usa

1. Usar `CRM` desde escritorio para alta de clientes, seguimiento comercial y analisis.
2. Usar `App Vendedores` en campo para visitas, checklists, fotos, audios y pedidos.
3. Revisar que cada cliente tenga asignacion comercial clara.
4. Sincronizar la operacion movil cuando vuelva la conectividad.
5. Consultar el historial del cliente desde la vista web para consolidar la informacion.

## Errores frecuentes

- Intentar hacer analisis profundo de cartera desde la app movil.
- Cargar visitas en CRM web cuando la operacion esta pensada para la app de vendedores.
- Confundir trabajo offline con falta de sincronizacion definitiva: revisar siempre el estado de subida pendiente.
