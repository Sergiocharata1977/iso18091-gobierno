## Que es

Es la parte del sistema donde cada proceso puede traducirse en objetivos de calidad, indicadores y mediciones periodicas.

## Para que sirve

Sirve para evaluar si un proceso cumple lo esperado, seguir tendencias y sostener decisiones de mejora con datos.

## Como se usa

1. Definir el proceso antes de crear objetivos o indicadores.
2. Crear el objetivo de calidad asociado al proceso.
3. Crear uno o mas indicadores que midan cumplimiento, eficacia o eficiencia.
4. Registrar mediciones con fecha, responsable y evidencia.
5. Revisar resultados para decidir acciones, correcciones o mejoras.

## Errores frecuentes

- Crear indicadores sin proceso asociado: luego no se pueden interpretar dentro del sistema.
- Medir sin frecuencia definida: los datos pierden comparabilidad.
- Cargar valores sin evidencia ni observacion: la medicion queda debil para auditoria o revision.
