## Que es

RRHH es el modulo donde se administra la estructura humana de la organizacion: personal, puestos, departamentos, competencias, capacitaciones y evaluaciones.

## Para que sirve

Sirve para mantener actualizada la ficha de cada empleado, definir que puesto ocupa, que competencias necesita y que formacion o evaluaciones respaldan ese rol dentro del sistema de gestion.

## Como se usa

1. Ingresar a `RRHH` desde el menu lateral.
2. Revisar el tablero general para detectar altas pendientes, capacitaciones o evaluaciones activas.
3. Usar `Personal` para crear o editar empleados.
4. Usar `Puestos` y `Departamentos` para ordenar la estructura organizacional.
5. Usar `Competencias`, `Capacitaciones` y `Evaluaciones` para registrar evidencia de desarrollo y desempeno.

## Errores frecuentes

- Crear personal sin puesto asignado: revisar primero el catalogo de puestos y departamentos.
- Cargar competencias sin relacionarlas con un puesto: esto dificulta medir brechas reales.
- Usar RRHH solo como legajo: el modulo tambien respalda evidencia para requisitos ISO vinculados a competencia y formacion.
