## Que es

Las Revisiones Periodicas Programadas permiten definir que elementos del SGC (procesos, indicadores, documentos, riesgos) deben ser revisados con una frecuencia determinada, y hacer seguimiento de esas revisiones.

## Para que sirve

Garantiza que el SGC se mantiene actualizado y que los requisitos de revision periodica de ISO 9001 (clausulas 9.1, 9.3) quedan cubiertos con evidencia trazable.

## Como se usa

1. Ingresar a **Revisiones programadas** desde el menu lateral.
2. Crear una revision indicando el elemento a revisar, frecuencia (mensual, trimestral, anual) y responsable.
3. El sistema genera automaticamente las fechas de proxima revision.
4. Cuando llega la fecha, el responsable recibe un aviso y debe completar la revision.
5. Al completar, se registra la fecha y el resultado quedando como evidencia.

## Errores frecuentes

- Definir revisiones sin responsable asignado: nadie recibe el aviso y la revision se omite.
- Revisar sin registrar observaciones: la evidencia queda incompleta para una auditoria.
